# Collision Detection @ Canvas

### Requirements

- node 8.x
- npm 5.x OR yarn 1.*

### Setup

- `$ git clone git@github.com:jacekk/collision-detection-on-canvas`
- `$ cd collision-detection-on-canvas`
- `$ yarn install`
- `$ yarn run start`
